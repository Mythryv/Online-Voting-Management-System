# Online-Voting-Management-System

## About the Project
An Online Voting Management System is a web-based application developed using HTML, CSS, and JavaScript. It allows users to vote for a candidate and instantly view the updated vote count. This project focuses on basic frontend development concepts such as DOM manipulation, event handling, and interactive user interfaces.

## Features
- Display multiple candidates with vote options
- Real-time vote count update
- Prevent multiple voting using simple logic
- Clean and simple user interface
- Interactive button-based voting system

## Technologies Used
- HTML – To structure the webpage
- CSS – To design and style the interface
- JavaScript – To handle voting logic and update vote counts dynamically

## How to Run
- Download or clone the repository.
- Open the project folder.
- Double-click index.html.
- The project will open in your browser.
